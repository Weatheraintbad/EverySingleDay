### 游戏内每天早晨都会赋予玩家一正一负的随机效果，并于聊天栏显示具体信息。
### Every morning in the game, players will be given a random effect of one positive and one negative, and specific information will be displayed in the chat bar.

  ### 效果一览
  ### Effect overview
  
| 宜/ Suitable for      | 效果/ Effect                                        | ID                 |
| --------------------- | ------------------------------------------------------- | ------------------ |
| 种植 / Farming          | 作物生长速度+50% / +50% crop growth speed                     | `farming`          |
| 战斗 / Combat           | 攻击力+20% / +20% attack damage                            | `combat`           |
| 运动 / Sprinting        | 移动速度+30% / +30% movement speed                          | `speed`            |
| 休养 / Resting          | 生命恢复+2倍 / +100% health regeneration                     | `health`           |
| 碰运气 / Gambling        | 幸运效果+钓鱼运气 / Bonus luck & fishing fortune                | `luck`             |
| 涅槃 / Rebirth          | 死亡时重生并爆炸 / Respawn on death with an explosion           | `phoenix`          |
| 收集 / Collecting       | 自动吸引附近物品 / Auto-attract nearby items                    | `magnet`           |
| 熬夜 / Staying-up       | 永久夜视效果 / Permanent night vision                         | `night_vision`     |
| 玩火 / Pyromania        | 免疫火焰伤害 / Immune to fire damage                          | `fire_immunity`    |
| 潜水 / Diving           | 水下呼吸+游泳速度 / Underwater breathing & faster swim          | `water_breathing`  |
| 战斗 / Vampire         | 攻击敌人时回复生命 / Heal when attacking foes                    | `vampire`          |
| 寻宝 / Treasure hunting | 发现隐藏宝箱和稀有矿石 / Reveal hidden chests & rare ores          | `treasure_hunter`  |
| 时间掌控 / Time control   | 时间流逝速度减半 / Halve the speed of time flow                 | `time_master`      |
| 偷袭 / Ambush           | 潜行时隐身+速度提升 / Invisibility + speed while sneaking        | `shadow_step`      |
| 战斗 / Combat           | 周期性获得随机元素保护 / Periodically gain random elemental shield | `elemental_shield` |

| 不宜/ Unsuitable for    | 效果/ Effect                                        | ID                 |
| --------------------- | ----------------------------------------------------- | ------------------ |
| 下水 / Entering water   | 在水中速度-50% / –50% speed in water                      | `water`            |
| 运动 / Sprinting        | 饥饿速度+2倍 / Hunger depletes 2× faster                   | `hunger`           |
| 战斗 / Combat           | 攻击力-20% / –20% attack damage                          | `weakness`         |
| 运动 / Sprinting        | 移动速度-20% / –20% movement speed                        | `slowness`         |
| 下矿 / Mining           | 挖掘速度-30% / –30% mining speed                          | `mining_fatigue`   |
| 碰运气 / Gambling        | 幸运效果-钓鱼运气差 / Reduced luck & poor fishing fortune      | `unluck`           |
| 死亡 / Death            | 死亡时产生爆炸 / Explode upon death                          | `explosive_death`  |
| 晒太阳 / Sunbathing      | 白天在阳光下持续受伤 / Continuous damage under daylight         | `sun_allergy`      |
| 外出 / Going outside    | 持续发出声音吸引怪物 / Constant noise draws monsters            | `noise_maker`      |
| 外出 / Going outside    | 方向感错乱（视角旋转） / Disorienting view spin                  | `confusion`        |
| 外出 / Going outside    | 持续吸引闪电 / Constantly attract lightning                 | `storm_maker`      |

### 目前支持的版本：

Fabric 1.20.1（需要 Fabric Loader 0.16.14 以及上版本，Fabric API任意版本）。
